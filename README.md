### Clone this repository

```bash
$ git clone https://github.com/Hemang14/Rentomate-App.git
```

### Go into the repository

```bash
$ cd Rentomate-App
```


### Install dependencies

```bash
$ flutter packages get
```

### Run the app

```bash
$ flutter run
```
